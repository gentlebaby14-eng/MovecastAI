# MoveCast AI

## 프로젝트 소개

MoveCast AI는 AI 기반 전국 도시 혼잡 예측 및 대응 의사결정 지원 플랫폼입니다.

시·도, 시·군·구, 장소 유형, 주요 거점, 날씨, 행사 정보, 방문객 수 등을 기반으로 도시 혼잡도를 예측하고, 공무원이 대응 계획을 세울 수 있도록 부서별 대응 방안을 제공합니다.

## 주요 기능

- 전국 혼잡 위험 지역 대시보드
- 시·도 → 시·군·구 → 장소 유형 → 주요 거점 4단계 선택
- 혼잡 위험도 예측
- 예상 유동인구 계산
- 대응 단계 산출
- 교통과, 안전총괄과, 홍보·민원 부서별 대응 추천
- 시민 알림 문구 자동 생성
- 사후 평가 및 AI 모델 개선 흐름 제시

## 사용 기술

- HTML
- CSS
- JavaScript

## 실행 방법

1. 이 리포지토리를 다운로드합니다.
2. `index.html` 파일을 브라우저에서 실행합니다.
3. 지역, 날씨, 행사 정보를 입력합니다.
4. `AI 혼잡 예측 실행` 버튼을 클릭합니다.
5. 예측 결과와 대응 방안을 확인합니다.
